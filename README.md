# Simple 3-Tier Web Application for Beginners

This repository contains a simple 3-tier web application designed for complete beginners. The application is built using HTML, CSS, JavaScript, and MySQL as the database.

## Project Structure

The project contains the following files and directories:

- `index.html`: Main HTML file for the web application.
- `style.css`: CSS file for styling the web pages.
- `script.js`: JavaScript file for adding interactivity to the web pages.
- `database.sql`: SQL script for creating the MySQL database schema and sample data.

## Prerequisites

Before running the application, ensure you have the following installed:

- A web browser (e.g., Google Chrome, Mozilla Firefox)
- MySQL database server
